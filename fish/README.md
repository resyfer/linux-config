# Fish Shell

- Install [Fish Shell](https://fishshell.com/)
- Install [Oh My Fish](https://github.com/oh-my-fish/oh-my-fish)
- Install Kawasaki Theme for Oh My Fish

```
omf install kawasaki
```

- For `config.fish`

```
cd ~/.config/fish/
nano config.fish
```

- For `init.fish`

```
$OMF_CONFIG
nano init.fish
```

Copy contents of `init.fish`
